# Gram-Sparsh   🌾 Gram Sparsh — Website Description
Company Kya Hai?
Gram Sparsh ek Indian consultancy firm hai jo teen main sectors mein kaam karti hai:

🍎 Food Technology — food processing, FSSAI compliance, product development
🌱 Agriculture — agri inputs, cold chain, FPO management
🤝 Grassroots / NGO Consultancy — NGO setup, CSR projects, rural development


🎨 Design & Theme

Rajasthani / Indian cultural theme — colors mein gold, saffron, red, teal use kiye hain
Top aur bottom pe colorful Rajasthani strip hai
Custom cursor hai (red dot + gold ring)
Corner ornaments aur jali pattern (lattice) background mein hai
Camel animation bottom pe chalta hai 🐪
Fonts: Montserrat (headings) + Inter (body)


📄 7 Slides / Pages
SlidePageS1Home — hero section with tagline & segment pillsS2About — company info, stats, values (Jharokha/arch style frame)S3Services — 3 segments: Food Tech, Agriculture, NGO workS4Platforms — Supplier marketplace, Job board, Academy coursesS5Blog/Insights — articles & knowledge resourcesS6Contact — inquiry form + contact cardsS7Thank You — closing page with footer

⚙️ Features

Full-page slide navigation (left/right arrows + dot nav)
Info modals — service cards click karo to detail popup aata hai
Form modals — supplier listing, job posting, course enrollment
3D tilt effect on service cards (mouse move se)
Jharokha parallax on About page
Toast notifications (form submit hone pe)
Mobile responsive — 900px aur 600px breakpoints pe layout adjust hota hai
Keyboard navigation — Escape se modals close hote hain


🛠️ Tech Stack

Pure HTML + CSS + Vanilla JavaScript — koi framework nahi
Google Fonts (CDN)
Sab kuch ek file mein — 1,215 lines
